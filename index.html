Langchain4j Lost and Found Agent

Project Summary:
An intelligent lost and found assistant system powered by AI that helps users register lost items, found items, and query for matches through natural language conversations. The system automatically identifies user intent and guides them through multi-turn dialogues to collect information. It features semantic search using RAG technology to intelligently match lost items with found items, providing an automated solution for lost and found management.

Tech Summary:
Built with Spring Boot 3.5.6 and LangChain4j 1.0.1, integrating Alibaba's Qwen-max LLM via OpenAI-compatible API. Uses Spring WebFlux for reactive streaming responses, MySQL with JPA/MyBatis-Plus for persistence, and implements RAG with text-embedding-v3 for semantic search. Features include Tool Calling for database access, structured JSON outputs, AOP-based chat flow management, and in-memory vector storage for similarity matching.
